## Virzz - CLI 命令行小工具

```
The Cyber Swiss Army Knife for terminal

Usage:
  virzz [flags]
  virzz [command]

Available Commands:
  ascii2str   String -> ASCII
  b32d        Base64 Decode
  b32e        Base32 Encode
  b58d        Base58 Decode
  b58e        Base58 Encode
  b64d        Base64 Decode
  b64e        Base64 Encode
  basex       Auto Base-X Decode
  bytes2hex   ByteString -> Hex
  bytes2str   ByteString -> String
  completion  Generate the autocompletion script for the specified shell
  dec2hex     Dec -> Hex
  githack     A `.git` folder disclosure exploit
  help        Help about any command
  hex2bytes   Hex -> Bytes String
  hex2dec     Hex -> Dec
  hex2str     Hex -> String
  str2ascii   String -> ASCII
  str2hex     String -> Hex
  urld        URL Decode
  urle        URL Encode
  version     Print the version

Flags:
  -D, --debug     Set Debug Mode
  -h, --help      help for virzz
  -V, --version   Show Version

Use "virzz [command] --help" for more information about a command.
```
