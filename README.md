## Virzz - CLI 命令行小工具

```
The Cyber Swiss Army Knife for terminal

Usage:
  virzz [flags]
  virzz [command]

Available Commands:
  alias       Print the version
  basex       Base 16/32/58/62/64/85/91/92/100 Encode/Decode
  basic       Some basic encodings

  domain      Some tools for Domain/SubDomain

  gopher      Generate Gopher Exp

  help        Help about any command

  md5         MD5 hash algorithm
  netlog      Netlog Server
  netool      Some net utils
  qrcode      A qrcode tool for terminal
  sha1        SHA1 hash algorithm
  sha224      SHA224 hash algorithm
  sha256      SHA256 hash algorithm
  sha384      SHA384 hash algorithm
  sha512      SHA512 hash algorithm
  sha512_224  SHA512/224 hash algorithm
  sha512_256  SHA512/256 hash algorithm
  version     Print the version

Flags:
  -h, --help      help for virzz
  -V, --version   Show Version

Use "virzz [command] --help" for more information about a command.
```
